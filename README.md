# Business Card OCR Web App

---

## Features

- Upload **one or multiple business card images**.
- **Preview** uploaded images before extraction.
- Extracts **text using OCR** in the browser (no server required).
- Generates **LinkedIn and Google search links** for contacts.
- Outputs results in a **structured, readable format**.

---

## Tech Stack

- **HTML / CSS / JavaScript**  
- **Tesseract.js** for OCR  
- Runs entirely **in the browser** (no Node.js required)
